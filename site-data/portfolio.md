## Project 1: Resume Chatbot
**Title:** Resume Chatbot: Harnessing ChatGPT for the Job Search
**Description:** A dashboard application for engaging with a customized chatbot agent hosted on Heroku. Uses retrieval-augmented generation (RAG) over documents including resume, CV, and publications. Session-based memory stored in AWS DynamoDB allows conversation review and corrections.
**Skills:** Python, LangChain, LLM (ChatGPT), AWS SAM, DynamoDB, Heroku, Dashboards
